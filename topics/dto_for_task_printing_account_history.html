  • Data objects

    Data object (DO) is an information entity that comprise details about all data and services that are needed to handle specific Step of a Functional task. Data object may be connected to more than one Step at the same time. Many Data objects may be connected to one Functional task as far as task may comprise several Steps.

    • Data object types

      Data object (DO) types are categories that were made by dividing all Functional tasks to an understandable logic groups. Data objects are divided into types and subtypes the same way as Functional tasks and Steps do to simplify their interconnection and relationship understanding.
